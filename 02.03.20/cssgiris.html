css'e giriş

htmlin kullanımı:

head etiketini daha çok class oluşturmakta kullanırız
class kavramı: seçilen etiketin hepsini etkilemeden seçtiğimiz sınıflara doğrudan etki etmeyi sağlar
  1. style özelliğiyle doğrudan şekil verme yoluyla
  2. head bloğu içerisinde
  3. link etiketi kullanarak harici bir css dosyasını lood bloğu içinde import edebiliriz.

html'in bakarkenki sıralaması

  1. etiket içi tanımlama
  2. dış veya iç stil tanımlama
  3. tarayıcı tarafından belirlenen, her tarayıcın belirlediği css çalışır.

syntax yapısı

head bloğunun içine yazdığımızda body içerisinde geçen bütün p etiketinin kapsadığı
içeriklere uyduracaktır.
p{color: red;}
p: seçilen etiketi ifade eder.
color: seçilebilecek özelliği ifade etmektedir.
red: seçilen özelliğe göre verilmek istenen değeri ifade eder.
